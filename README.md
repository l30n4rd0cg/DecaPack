# DecaPack
Minecraft Datapack made for the Decacraft SMP.

## Installation

### When Creating a Minecraft World
1. Go to `More > Datapacks` and click `Open Pack Folder`.
2. Drag the `.zip` file in that folder.
3. If 2 doesn't work, uncompress the `.zip` and use that.

### Installing on an existing Minecraft World
1. Uncompress the `.zip` file in the `(.minecraft)/saves/<World Name/Folder>/datapacks/` folder.
2. Execute `/reload`

### Melius Commands Mod Setup
1. Install the Melius Commands Mod (link of Modrinth [here](https://modrinth.com/mod/melius-commands)).
2. Drag/Move the datapack decapack.json file to `(.minecraft)/config/melius-commands/commands/`.
3. Execute `/reload`

## Usage

### /shop
Opens the shop.

### /sell <price>
Sell the held item for the specified price.

### /money <get|set> <player> <amount: set>
#### /money get <player>
Get the amount of money the specified player has.
### /money set <player> <amount> [OP ONLY]
Set the amount of monet the specified player has.

### /pay <player> <amount>
Pay some amount of money to the specified player.
